# Enterprise Network Project (Basic)

## Project Overview
Designed and configured a small enterprise network using Cisco Packet Tracer. The project simulates a business environment with multiple departments, secure network segmentation, automated IP addressing, and wireless connectivity.

## Network Features
- VLAN segmentation for:
  - Admin
  - Marketing
  - Finance
  - Reception
- Inter-VLAN Routing using Router-on-a-Stick
- 802.1Q Trunking between switches and router
- DHCP configuration for automatic IP address assignment
- Wireless Access Point configuration for client connectivity
- Basic network troubleshooting and connectivity testing

## Technologies & Tools
- Cisco IOS CLI
- VLANs
- DHCP
- 802.1Q Trunking
- Router-on-a-Stick
- Wireless Networking

## Skills Demonstrated
- VLAN creation and port assignment
- Switch configuration
- Router interface configuration
- IP addressing and subnetting
- DHCP pool configuration
- Network troubleshooting using ping and CLI commands
